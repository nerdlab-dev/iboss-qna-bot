# 아이보스 질문답변 봇 아키텍처

## 시스템 구성

### 1. 크롤링 모듈
- **기술**: Selenium WebDriver + BeautifulSoup4
- **대상**: https://www.i-boss.co.kr/ab-2109 (아이보스 질문답변 게시판)
- **특징**: 
  - 동적 페이지 렌더링 대응
  - 최신 10개 질문 수집
  - 게시물 ID 기반 중복 체크

### 2. 디스코드 연동
- **라이브러리**: discord.py
- **기능**:
  - Embed 형식의 깔끔한 메시지
  - 최대 3개 질문 상세 표시
  - 추가 질문 개수 안내
  - 채널별 권한 체크

### 3. 스케줄링
- **macOS**: LaunchAgent (권장)
- **Linux**: Cron 또는 Systemd
- **실행 시간**: 매일 오전 9시

### 4. 데이터 저장
- **last_qna_posts.json**: 마지막 수집한 게시물 ID 저장
- **logs/**: 실행 로그 저장

## 실행 흐름

1. 봇 시작 및 Discord 연결
2. 아이보스 질문답변 페이지 크롤링
3. 이전 게시물 ID와 비교하여 새 질문 필터링
4. 새 질문이 있으면 Discord 채널에 전송
5. 현재 게시물 ID 저장
6. 종료

## 보안 고려사항

- `.env` 파일로 민감한 정보 분리
- Discord 토큰 환경변수 관리
- 채널 접근 권한 검증
